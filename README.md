# docker-shell-helper

## Install 
    git clone git@github.com:remibodin/docker-shell-helper.git
    export PATH=${PATH}:${PWD}/docker-shell-helper

## Commandes

| cmd          |                           |
|--------------|---------------------------|
| **dsh-rmsc** | remove stopped containers |
| **dsh-rmui** | remove unstagge images    |
